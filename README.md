# Senior Care Program

A Java-based application developed in Grade 12 to simulate a basic senior care management system.  
The program allows users to interact with and manage information related to seniors through a simple interface, demonstrating object-oriented programming concepts and control flow logic.

---

## Features
- Menu-driven user interaction
- Management of senior care–related data
- Input validation and conditional logic
- Modular design using classes and methods

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Command-line interface

---
